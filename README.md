# NoCapEdit (nce)

> [!IMPORTANT]
> **NoCapEdit — 書くことを邪魔しないメモエディタ**
> 
> 起動直後からすぐに書き始められ、書いたものがすぐに保存されます。<br>
> 保存することを意識せず、思いついたことを、すぐに書き留められます。<br>
> あなたの **書くことを邪魔しない** 、シンプルで洗練された **メモエディタ** です。

一般的なテキストエディタで発生する「保存時にファイル名や保存場所を指定する」という手間をなくし、書くことに集中できるようになります。

起動時やタブ切り替え時、設定変更後もカーソル位置や選択状態、フォーカスが常に復元・維持されます。

ストレスフリーでシームレスなエディタ環境を、ぜひ体感してください。

## 📦 インストール方法

用途やユーザー環境に合わせて以下から選択してください。

- **[ポータブル版（ZIP形式 / インストール不要）](https://github.com/yasudajs/NoCapEdit/releases/download/v0.1.35/NoCapEdit_v0.1.35_x64_portable.zip)**
  - ダウンロードしたフォルダを解凍するだけで、すぐに使い始められます。USBメモリなどに入れて持ち運ぶことも可能です。
- **[標準インストーラー（EXE形式 / 推奨）](https://github.com/yasudajs/NoCapEdit/releases/download/v0.1.35/NoCapEdit_0.1.35_x64-setup.exe)**
  - パソコンの個人用エリアにインストールするため、管理者権限（パスワード入力等）なしで簡単に導入できます。迷ったらこちらがおすすめです。
- **[システム全体用インストーラー（MSI形式）](https://github.com/yasudajs/NoCapEdit/releases/download/v0.1.35/NoCapEdit_0.1.35_x64_ja-JP.msi)**
  - 学校や職場などで、1台のパソコンを複数のユーザーで使う場合に適しています。（※インストールには管理者権限が必要です）

> [!IMPORTANT]
> **インストール時の警告画面について**
> 本アプリは個人開発のためデジタル署名を行っておりません。そのため、インストール時にWindowsから警告が表示される場合がありますが、安全にご利用いただけます。
> - **「WindowsによってPCが保護されました」と表示された場合**: 画面内の **「詳細情報」** をクリックし、表示された **「実行」** ボタンを押して進めてください。（※EXE版はこの警告が出る場合があります）
> - **「このアプリがデバイスに変更を加えることを許可しますか？」と表示された場合**: **「はい」** を選択して進めてください。

## 🚀 主な特徴

### 💾 保存モードとファイル生成

1. **2つの保存モード（自動保存 / 手動保存）**
   - **自動保存モード（デフォルト）**: 文字の入力が停止してから **3秒後** に自動保存を行います。
   - **手動保存モード**: 自動では保存を行わず、`Ctrl + S` による手動操作時のみ保存を行います。設定画面からいつでも切り替え可能です。
   - 保存状態の更新は画面左下（ステータスバー）のテキスト（「編集中...」「保存済み」）のみで静かに表示され、ポップアップや通知など作業の集中を妨げる表示は一切行われません。
   - 手動保存モードで編集中にタブを閉じたりウィンドウを閉じた場合、**確認のメッセージは出ず、編集中の内容は破棄されます**ので、ご注意ください。
   - ファイル編集中にモードを切り替えた場合、開いているファイルにもモード変更が適用されます。

2. **自動保存モード時のファイル管理**
   - **文字を入力し始めてからファイルを作る機能**: 新しいタブを開いた（またはアプリを起動した）だけでは、まだパソコン内にファイルは作られません。文字を入力して最初の保存が行われたタイミングで、その時の日時を名前としたファイル（例: `20260713_184500.nctx`）が初めて自動作成されます。これにより、何も書いていない空のファイルが勝手にフォルダに増えるのを防ぎます（すでに同じ名前のファイルがある場合は、自動的に `_01`, `_02` のような連番が名前の後ろに付きます）。
   - **空になったファイルを自動で整理する機能**: 文字を何も入力しなかったタブや、スペース・改行だけを入力したタブを閉じたときは、余計なファイルが残らないように自動で整理（削除）されます。これにより、後から「中身が空っぽの不要なファイル」を自分で削除する手間がなくなります。

### 🔒 データの安全性とカスタマイズ

3. **ファイルの安全な保存（保存エラーやファイル破損の防止）**
   - **オンラインストレージへの対応**: OneDriveなどのオンラインストレージ（同期フォルダ）内でも、保存エラーやファイルの破損が起こりにくい安全な保存方式を採用しています。
   - **重ねて操作しても安心**: 文字の入力中にタブを切り替えたり、アプリを閉じたりする操作が重なっても、保存の処理が衝突してファイルが壊れないように設計されています。

4. **テーマの切り替え（ダーク / ライト / ソフトダーク）**
   - 3種類のデザインテーマ（ダーク、ライト、目に優しいソフトダーク）の切り替えに対応しています（タイトルバーの色も連動します）。
   - 設定はアプリを再起動した後もそのまま維持されます。

5. **フォント・表示のカスタマイズ**
   - `Ctrl` + `マウスホイール` または `Ctrl` + `+` / `-` によるフォントの拡大縮小機能。
   - `Ctrl` + `Shift` + `マウスホイール` または `Ctrl` + `Shift` + `+` / `-` による行間の拡大縮小機能。
   - **等幅フォント**（文字の横幅がそろっていて、プログラムやメモが整然と読みやすくなるフォント）と、それ以外のフォントを分けて選ぶことができます。

### ⚙️ バージョンと設定の管理

6. **バージョン表示および自動アップデート通知**
   - アプリケーションのタイトルバーに現在のバージョン番号が `NoCapEdit [ Ver a.b.c ]` の形式で表示され、現在のバージョンをひと目で確認できます。
   - 起動時に新しいバージョンがリリースされているかをバックグラウンドで自動チェックし、最新版がある場合はタイトルバーおよび設定画面内にアップデート通知（リリースノートへのリンク付き）を表示します。

7. **データの保存場所**
   - **作成したメモの保存先**: 「ドキュメント」フォルダ内の `nce` というフォルダに自動で保存されます。
   - 初回起動時や、設定された保存先フォルダが見つからない場合は、起動時に「保存先フォルダを選ぶ画面」が表示されます。

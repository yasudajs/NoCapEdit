# NoCapEdit (nce)

NoCapEdit は、起動直後からすぐに書き始められ、ファイルの保存を意識することなく使えるようにした、シンプルで洗練された **メモエディタ** です。

一般的なテキストエディタで発生する「保存時にファイル名や保存場所を指定する」という手間をなくし、書くことに100%集中できるようになります。

起動時やタブ切り替え時、設定変更後もカーソル位置や選択状態、フォーカスが常に復元・維持されます。

ストレスフリーでシームレスなエディタ環境を、ぜひ体感してください。

## 📦 インストール方法

用途やユーザー環境に合わせて以下から選択してください。

- **[ポータブル版（ZIP形式）](https://github.com/yasudajs/NoCapEdit/releases/download/v0.1.30/NoCapEdit_v0.1.30_x64_portable.zip)**
  - インストール不要。ダウンロード後に展開（解凍）するだけで即実行できます。
- **[NSIS インストーラー（EXE形式 / 推奨）](https://github.com/yasudajs/NoCapEdit/releases/download/v0.1.30/NoCapEdit_0.1.30_x64-setup.exe)**
  - 画面の指示に従って、設定を確認しながらインストールしたい方向けです。個人フォルダ（AppData）にインストールされるため、管理者権限なしでも手軽に導入できます。迷ったらこちらをお選びください。
- **[MSI インストーラー](https://github.com/yasudajs/NoCapEdit/releases/download/v0.1.30/NoCapEdit_0.1.30_x64_ja-JP.msi)**
  - 画面遷移を省いて、静かに素早くインストールしたい方向けです。システム全体（Program Files）にインストールされるため、実行時に管理者権限が必要です。

> [!IMPORTANT]
> **インストール時の警告画面について**
> 本アプリは個人開発のためデジタル署名を行っておりません。そのため、インストール時にWindowsから警告が表示される場合がありますが、安全にご利用いただけます。
> - **「WindowsによってPCが保護されました」と表示された場合**: 画面内の **「詳細情報」** をクリックし、表示された **「実行」** ボタンを押して進めてください。（※EXE版はこの警告が出る場合があります）
> - **「このアプリがデバイスに変更を加えることを許可しますか？」と表示された場合**: **「はい」** を選択して進めてください。

## 🚀 主な特徴

### 💾 保存モードとファイル生成

1. **2つの保存モード（自動保存 / 手動保存）**
   - **自動保存モード（デフォルト）**: 文字の入力が停止してから **3秒後** に自動保存を行います。
   - **手動保存モード**: 自動では保存を行わず、`Ctrl + S` による手動操作時のみ保存を行います。設定画面からいつでも切り替え可能です。
   - 保存状態の更新はステータスバー左下のテキスト（「編集中...」「保存済み」）のみで行われ、ポップアップや通知など作業の集中を妨げる表示は一切行われません。
   - 手動保存モードで編集中にタブを閉じたりウィンドウを閉じた場合、**確認のメッセージは出ず、編集中の内容は破棄されます**ので、ご注意ください。
   - ファイル編集中にモードを切り替えた場合、開いているファイルにもモード変更が適用されます。

2. **自動保存モード時のファイル管理**
   - **起動時・タブ作成時のファイル遅延作成**: アプリの起動時および新規タブ作成時には、ディスク上にファイルの実体を作成しません。ただし、ファイル名に使用されるタイムスタンプは、タブ作成時（アプリ起動時または新規タブ追加時）の時刻が保持されます。ユーザーが文字を入力し、最初の保存処理（自動・手動）が実行されたタイミングで、その保持されたタイムスタンプを基にした `YYYYMMDD_HHMMSS.nctx` 形式のテキストファイルが初めて生成されます（初回保存時に同名ファイルが存在する場合は自動的に `_01`, `_02` のように連番を付加します）。これにより、無駄なファイル作成や不要なディスクI/Oを防止しつつ、作成時点の時刻に基づいた一貫したファイル管理を実現します。
   - **空白タブの自動クリーンアップ（未作成スキップ）**: 文字を何も入力しないまま（半角/全角スペース、改行、タブのみの「見えない文字のみ」の状態）でタブを閉じるか、アプリを終了した場合、ディスクにすでに作成されているファイルであれば自動的に削除されます。まだファイルが作成されていない場合（文字を一度も入力していない、または入力後に全削除して閉じた場合）は、ディスク操作を行わずそのまま安全に閉じられます。これにより、不要な空ファイルがフォルダに溜まることを防ぎます。

### 🔒 データの安全性とカスタマイズ

3. **安全保存方式と保存ロック（排他制御）**
   - **安全保存方式（手動・自動共通）**: ファイル保存時は、一度一時ファイル (`.tmp`) に書き出してからリネーム置換する安全な方式を採用しています。OneDrive 等の同期フォルダ内でのファイルロック競合による破損を防ぎます。
   - **保存ロック（自動保存時）**: 自動保存中にタブの切り替えやアプリ終了が重なった場合でも、同じファイルに対して同時に書き込みが走らないよう、タブごとに非同期処理の排他制御ロック（`isSaving`）を導入しています。

4. **テーマの切り替え（ダーク／ライトモード）**
   - ダークモードとライトモードの切り替えに対応（タイトルバー色も連動）。
   - 設定はアプリ再起動後も維持されます。

5. **フォント・表示のカスタマイズ**
   - `Ctrl` + `マウスホイール` または `Ctrl` + `+` / `-` によるフォントの拡大縮小機能。
   - `Ctrl` + `Shift` + `マウスホイール` または `Ctrl` + `Shift` + `+` / `-` による行間の拡大縮小機能。
   - 等幅フォント（Monospace）とその他フォントを分類して選択できるフォントファミリー切り替え機能。

### ⚙️ バージョンと設定の管理

6. **バージョン表示および自動アップデート通知**
   - アプリケーションのタイトルバーに現在のバージョン番号が `NoCapEdit [ Ver a.b.c ]` の形式で表示され、現在のバージョンをひと目で確認できます。
   - 起動時に新しいバージョンがリリースされているかをバックグラウンドで自動チェックし、最新版がある場合はタイトルバーおよび設定画面内にアップデート通知（リリースノートへのリンク付き）を表示します。

7. **データと設定の保存場所**
   - **設定ファイル**: `%APPDATA%/NoCapEdit/config.json`
   - **デフォルトの保存先 (ホームフォルダ)**: `C:/Users/<ユーザー名>/Documents/nce`
   - 初回起動時、または設定されたフォルダが見つからない場合は、起動時にホームフォルダ選択ダイアログが表示されます。

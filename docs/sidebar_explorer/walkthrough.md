# ウォークスルー

サイドバー（ファイルエクスプローラ）から直接ファイル・フォルダの作成、名前変更、通常削除（ゴミ箱移動）を行える機能を実装しました。

## 実装された変更

### 1. バックエンド (Rust)
* **[Cargo.toml](file:///c:/work/NoCapEdit/Cargo.toml)**: ゴミ箱移動機能のために `trash` クレートを追加し、Tauri allowlist の変更に伴い `dialog-ask` feature を有効化しました。
* **[main.rs](file:///c:/work/NoCapEdit/src/main.rs)**:
  * `create_file_or_dir`: セキュリティ検証を行った上で、ディスク上に新規ファイル/フォルダを作成します。同名衝突が発生した場合は、ファイルなら `[stem]_[連番].[ext]`、フォルダなら `[名前]_[連番]` で自動回避します。
  * `rename_file_or_dir`: 対象の名前を変更します。すでに同名要素が存在する場合はエラーを出します。
  * `trash_file_or_dir`: OSのゴミ箱に対象を移動します。
  * `tauri.conf.json` での allowlist `dialog.ask` の有効化。

### 2. フロントエンド
* **[index.html](file:///c:/work/NoCapEdit/src/dist/index.html)**: カスタムコンテキストメニューの DOM 要素（`#contextMenu`）を配置しました。
* **[style.css](file:///c:/work/NoCapEdit/src/dist/style.css)**: テーマ追従可能なコンテキストメニューの表示スタイルおよびツリー内のインライン編集用テキストボックス（`.tree-input`）のスタイルを追加しました。
* **[main.js](file:///c:/work/NoCapEdit/src/dist/main.js)**:
  * 右クリックした対象（ファイル、フォルダ、または背景）を判別し、コンテキストメニュー項目を動的に切り替えて表示する制御を実装しました。
  * 「新規作成」「名前変更」の際に、ツリーの対象ノードをインライン入力欄に置き換える処理を追加しました。
    * 拡張子なしでの確定時は自動的に末尾へ `.nctx` を補完する仕様を実装しました。
    * 新規ファイル作成時は、自動的にそのファイルをエディタの新規タブとしてオープンします。
  * 「削除」実行時に Tauri ダイアログによる確認を表示し、承認時に `trash` クレートを介してゴミ箱に移動させます。
  * **エディタ連動**:
    * 開いているファイルを削除した際、エディタ上の対象タブを自動で（保存確認を挟まず）強制クローズします。
    * 名前変更の際、対象タブのファイル名・パス情報を新しいものへ動的に書き換えます。

## 検証結果

### ビルドの確認
* `cargo check` および `cargo build` を実行し、コンパイルが正常に通ることを確認しました。

### 手動テスト手順 (ユーザー様での確認用)
1. **アプリ起動**: `cargo run` などでアプリを起動します。
2. **コンテキストメニュー**:
   * サイドバーのファイルやフォルダ、または何もない余白を右クリックし、コンテキストメニューが適切な項目（ファイルは「名前変更」「削除」、フォルダは「新規ファイル」「新規フォルダ」「名前変更」「削除」、背景は「新規ファイル」「新規フォルダ」）で表示されるか確認します。
   * メニュー外クリックや `Esc` キーで閉じるか確認します。
   * テーマ（Dark/Soft Dark/Light）を切り替えてもメニューデザインが崩れないか確認します。
3. **新規作成**:
   * 「新規ファイル作成」を選択し、インライン入力欄に入力します。
   * 空欄のまま Enter を押して `名称未設定.nctx` が作成され、自動的に新規タブとして開くことを確認します。
   * 拡張子なしで入力（例: `memo`）して確定し、`memo.nctx` として作成されるか確認します。
   * フォルダ作成でも同様に `新しいフォルダ` などのインライン作成・衝突回避が働くか確認します。
4. **名前変更**:
   * ファイルを右クリックして「名前を変更」し、インラインで変更します。
   * そのファイルが開かれている場合、タブ名が追従して更新されるか確認します。
5. **削除**:
   * ファイルを右クリックして「削除」を選択し、ダイアログで「はい」を押します。
   * OSのゴミ箱に移動したか確認します。
   * 開いていたタブが自動で閉じられるか確認します。

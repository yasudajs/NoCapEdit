# ⌨️ キーボード操作 & ショートカット一覧 (Shortcuts)

NoCapEdit で利用できるキーボードショートカットとマウス操作の一覧です。
キーボードのみで効率的に執筆やファイル操作を行えるように設計されています。

> [!TIP]
> アプリケーションの基本的な画面構成や使い方については、[ユーザーガイド](USER_GUIDE.md) をご参照ください。

---

## 📝 エディタ操作
テキスト入力・編集エリアでの操作です。

| ショートカットキー | 操作内容 | 備考 |
| :--- | :--- | :--- |
| `Ctrl + S` | **手動保存** | 自動保存モード時でも、即座にディスクへ書き込みます。 |
| `Ctrl + +` / `-` <br> `Ctrl + マウスホイール` | **文字サイズの拡大・縮小** | エディタの文字サイズを変更します（8pt 〜 72pt）。 |
| `Ctrl + Shift + +` / `-` <br> `Ctrl + Shift + マウスホイール` | **行間の拡大・縮小** | エディタの行間を変更します（1.0倍 〜 3.0倍）。 |
| `Tab` | **インデントの挿入** | 設定で選択されたインデント（タブ文字、スペース2/4文字）を挿入します。<br>複数行を選択している場合は、選択したすべての行頭にインデントを挿入します。 |
| `Shift + Tab` | **インデントの削除** | 選択行（または現在行）の行頭からインデントを1段階削除します。 |

---

## 🗂 タブ操作
開いているファイルの切り替えや操作です。

| ショートカットキー | 操作内容 | 備考 |
| :--- | :--- | :--- |
| `Ctrl + Tab` | **右隣 of タブへ移動** | 一番右端のタブで押した場合は、先頭（一番左）のタブに戻ります。 |
| `Ctrl + Shift + Tab` | **左隣 of タブへ移動** | 先頭（一番左）のタブで押した場合は、末尾（一番右）のタブに戻ります。 |

---

## 📁 ファイルツリー操作
画面左側のファイルエクスプローラ（ファイルツリー）にフォーカスが当たっている時の操作です。
（※ `Ctrl + E` でいつでもフォーカスをツリーに移すことができます）

| ショートカットキー | 操作内容 | 備考 |
| :--- | :--- | :--- |
| `Ctrl + E` | **ファイルツリーとエディタの行き来（トグル）** | ツリーが非表示の場合は開き、最後に選択された項目にフォーカスします。ツリーでファイルを選択中に押すと、そのファイルを開きつつツリーを閉じてエディタへ戻ります。 |
| `Esc` | **エディタへフォーカスを戻す** | **ツリーは開いた状態を維持**したまま、操作を終了して文字入力へ戻ります。<br>※ファイルを「切り取り」中の場合は、切り取り状態をキャンセルします。 |
| `↑` / `↓` （矢印キー） | **項目の選択移動** | 現在表示されている（展開されている）ファイルを上下に移動して選択します。 |
| `→` （矢印キー） | **フォルダの展開** | 閉じているフォルダを展開します。すでに展開されている場合は何もしません。 |
| `←` （矢印キー） | **フォルダの折りたたみ / 親フォルダへ移動** | 展開しているフォルダを折りたたみます。すでに折りたたまれている場合、またはファイルを選択している場合は、その親フォルダのアイテムへ選択が移動します。 |
| `Enter` | **ファイルを開く / フォルダ開閉のトグル** | ファイルを選択している場合はエディタで開き、自動でエディタへフォーカスを移動します。フォルダの場合は展開/折りたたみを切り替えます。 |
| `Ctrl + N` | **新規ファイルの作成** | 選択中のフォルダ配下（ファイル選択時はその親フォルダ内）に新規ファイルのインライン入力欄を表示します。 |
| `Ctrl + D` | **新規フォルダの作成** | 選択中のフォルダ配下（ファイル選択時はその親フォルダ内）に新規フォルダのインライン入力欄を表示します。 |
| `F2` | **名前の変更** | 選択中のファイル・フォルダに名前変更用のインライン入力欄を表示します。 |
| `Ctrl + X` | **アイテムの切り取り** | 選択中のファイル・フォルダを移動元として記憶します。対象アイテムはツリー上で半透明（50%透明）で表示されます。 |
| `Ctrl + C` | **アイテムのコピー** | 選択中のファイル・フォルダをコピー元として記憶します。 |
| `Ctrl + V` | **貼り付け（コピー・移動の実行）** | 記憶したアイテムを選択フォルダ配下に貼り付けます。同名衝突時は自動で末尾に連番（`_01` など）が付きます。 |
| `Delete` | **通常削除（ごみ箱へ移動）** | 確認ダイアログを表示し、承諾後にOSのごみ箱へ移動します。※空でないフォルダは削除できません。 |
| `Shift + Delete` | **完全削除（物理削除）** | 確認ダイアログを表示し、承諾後にディスクから永久削除します（ごみ箱を経由しません）。※空でないフォルダは削除できません。 |

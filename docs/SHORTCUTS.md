# ⌨️ キーボード操作 & ショートカット一覧 (Shortcuts)

NoCapEdit で利用できるキーボードショートカットとマウス操作の一覧です。
キーボードのみで効率的に執筆やファイル操作を行えるように設計されています。

> [!TIP]
> アプリケーションの基本的な画面構成や使い方については、[ユーザーガイド](USER_GUIDE.md) をご参照ください。

---

## 📝 エディタ操作
テキスト入力・編集エリアでの操作です。

| ショートカットキー | 操作内容 | 備考 |
| :--- | :--- | :--- |
| `Ctrl + S` | **手動保存** | 自動保存モード時でも、即座にディスクへ書き込みます。 |
| `Ctrl + +` / `-` <br> `Ctrl + マウスホイール` | **文字サイズの拡大・縮小** | エディタの文字サイズを変更します（8pt 〜 72pt）。 |
| `Ctrl + Shift + +` / `-` <br> `Ctrl + Shift + マウスホイール` | **行間の拡大・縮小** | エディタの行間を変更します（1.0倍 〜 3.0倍）。 |
| `Tab` | **インデントの挿入** | 設定で選択されたインデント（タブ文字、スペース2/4文字）を挿入します。<br>複数行を選択している場合は、選択したすべての行頭にインデントを挿入します。 |
| `Shift + Tab` | **インデントの削除** | 選択行（または現在行）の行頭からインデントを1段階削除します。 |

---

## 🗂 タブ操作
開いているファイルの切り替えや操作です。

| ショートカットキー | 操作内容 | 備考 |
| :--- | :--- | :--- |
| `Ctrl + Tab` | **右隣のタブへ移動** | 一番右端のタブで押した場合は、先頭（一番左）のタブに戻ります。 |
| `Ctrl + Shift + Tab` | **左隣のタブへ移動** | 先頭（一番左）のタブで押した場合は、末尾（一番右）のタブに戻ります。 |

# main.js リファクタリング (ES Modules分割)

## 概要
フロントエンドの肥大化した `main.js`（約2200行）を、機能・責務ごとに複数の ES Modules に分割し、今後の機能追加や多言語対応を安全に行えるようにリファクタリングを実施しました。

## 実装内容
- `src/dist/js/` ディレクトリを新設し、以下のモジュールを作成しました。
  - `state.js` : アプリケーションのグローバルな状態（`appState`、DOMキャッシュ等）の管理
  - `utils/helpers.js` : パス操作やバージョン比較などのユーティリティ関数
  - `core/tauri.js` : Tauri API のラッパー
  - `core/fileSystem.js` : ファイル保存、読み込み、自動保存などのロジック
  - `ui/tabs.js` : タブの作成、切り替え、閉鎖処理
  - `ui/editor.js` : エディタの入力監視、フォントサイズ・行間の調整処理
  - `ui/sidebar.js` : エクスプローラの描画、ファイル操作コンテキストメニューの処理
  - `ui/settings.js` : 設定モーダルの開閉、テーマ切り替え、フォント切り替えの処理
  - `main.js` : アプリのエントリポイント。各モジュールを結合し、起動時のイベントリスナー登録と初期化を実施
- 既存の `src/dist/main.js` を削除しました。
- `src/dist/index.html` において、 `<script type="module" src="js/main.js"></script>` を読み込むように変更しました。

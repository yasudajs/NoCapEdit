# 📖 NoCapEdit ユーザーガイド (User Guide)

NoCapEdit の基本的なコンセプト、画面の構成、および各機能の使い方について解説するガイドブックです。

---

## 💡 コアコンセプト
NoCapEdit は、**「ファイル名や保存場所の入力に気を取られず、書くことに集中できる」** ことを最優先に設計されたエディタです。

### 1. 自動保存モード (オートセーブ)
文字を入力し、タイピングが止まってから **3秒後** に自動でファイルに保存されます。自分で「上書き保存」をする必要はなく、突然アプリを閉じてもデータが消える心配はありません。

### 2. 遅延ファイル作成
新規タブを開いた時点では、ディスク上にファイルの実体は作成されません。**「最初の文字を入力した」タイミング** で、初めてファイル（`yyyymmdd_hhmmss.nctx`）が作成されます。中身の無い空のファイルが勝手にフォルダに増えてしまうのを防ぎます。

### 3. 空ファイルの自動整理
何も入力していない空のタブ（またはスペースや改行しか入力していないタブ）を閉じる際、そのファイルを自動でディスク上からクリーンアップ（削除）します。不要なメモのゴミがフォルダに溜まりません。

## 🖥️ 画面構成
NoCapEdit のウィンドウは、大きく分けて以下の4つの領域で構成されています。

```
+-----------------------------------------------------------+
| [🗂] |  未保存1 ×  |  2026/07/16 18:00:00 ×  | [+]      [⚙️] | <-- ① トップバー (タブ / 設定)
+---------+-------------------------------------------------+
|         |                                                 |
| 📁 nce  |  ここに入力して記述します。                     |
|         |  思いついたことをそのまま書き留めてください。   |
| 📄 a.txt|                                                 | <-- ② メインコンテンツ
| 📄 b.txt|                                                 |     (左: サイドバー / 右: エディタ)
|         |                                                 |
+---------+-------------------------------------------------+
| 準備完了                                 Ln 1, Col 1 | 0  | <-- ③ ステータスバー
+-----------------------------------------------------------+
```

### ① トップバー（タブ領域）
- **左端の [🗂] ボタン**: サイドバー（ファイルエクスプローラ）の開閉を切り替えます。
- **タブ一覧**: 開いているファイルをタブで切り替えます。
- **[+] ボタン**: 新規ファイル（遅延作成待ち状態）を新しいタブで開きます。
- **右端の [⚙️] ボタン**: 設定画面（右ドック）の開閉を切り替えます。

### ② サイドバー（ファイルエクスプローラ）
- 設定されたホームフォルダ（デフォルトは `Documents/nce`）の中身がツリー構造で表示されます。
- ファイルやフォルダをクリックすることで、展開/折りたたみ、ファイルオープン、各種ファイル操作（新規作成、名前変更、ドラッグ＆ドロップでの移動、削除など）が行えます。

### ③ エディタ領域
- メインのテキスト入力エリアです。
- 文字コードは `UTF-8`、改行コードは `CRLF` に自動的に統一・保存されます。

### ④ ステータスバー（画面下部）
- **左側**: 保存の状態（「編集中」「保存中...」「保存済み」）を表示します。
- **右側**: 現在のカーソルの行・列位置、文字数（範囲選択時は選択文字数も）、フォントサイズ、行間設定を表示します。

## ⚙️ 各種機能と設定

### 1. 保存モードの変更（自動保存 / 手動保存）
設定ドック（右上の歯車ボタン）から、保存モードを「自動保存（推奨）」または「手動保存」に切り替えることができます。
- **手動保存モード**にすると、自動保存は一切行われず、`Ctrl + S` による保存操作時のみ書き込みが実行されます。
- *注意*: 手動保存モードでは、保存せずにタブやアプリを閉じた際、確認メッセージを表示せず内容がそのまま破棄されます。

### 2. デザインテーマの切り替え
設定ドックの「テーマ」から、以下の3つに切り替えることができます。タイトルバーの色も連動します。
- **ダーク**: 完全な黒背景ベースの引き締まったダークモード。
- **ソフトダーク**: グレー背景ベースの、目に優しい柔らかいダークモード。
- **ライト**: 白背景ベースのすっきりしたライトモード。

### 3. 文字数カウント方法の設定
設定ドックから、ステータスバーに表示される文字数のカウント方法を「改行を含む（デフォルト）」または「文字数のみ（改行を除く）」から選択できます。

### 4. インデントの挙動設定
エディタ上で `Tab` キーを押したときに挿入される文字を、「タブ文字（`\t`）」「スペース2文字」「スペース4文字」から選択できます。

## ⌨️ キーボード操作について
NoCapEdit は、キーボードのみでファイルの作成や削除、移動、タブ切り替えなど、すべての主要操作を完結できるように作られています。

詳しいショートカットキーの組み合わせは、**[ショートカットキー一覧 (SHORTCUTS.md)](SHORTCUTS.md)** をご覧ください。

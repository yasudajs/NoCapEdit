  ## リファクタリング中に発見した不具合および足りない要件

  - Ctrl+E でエディタからファイルツリーを開くことができるが、ファイルツリーを閉じることが出来ない。
    - ファイルツリーにフォーカスがあるときは、Ctrl+Nでツリーの開閉をトグルしてはどうか？
  - ホームフォルダで、Ctrl + N / D をして、確定する前に Esc でキャンセルした時、ホームフォルダにフォーカスが残らず、リスト最上位にフォーカスが移るので、2回目以降にCtrl + N / D をすると、最上位のフォルダでの操作になってしまう。
    - キャンセルした場合、キャンセルした階層のファイルもしくや親フォルダにフォーカスが移動するような動作に修正する必要がある。

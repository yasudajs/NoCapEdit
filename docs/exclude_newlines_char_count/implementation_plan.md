# 文字数カウントの改行除外対応

ステータスバーに表示される文字数カウントについて、改行文字（`\n` や `\r\n`）を文字数に含めない「純粋な文字数」として計算・表示するように変更します。

## 変更内容

### `src/dist/main.js`

- `updateEditorMetrics()` 関数において、エディタのテキスト全体の長さを取得している箇所を修正します。
- 現在: `const chars = value.length;`
- 変更後: `const chars = value.replace(/\r?\n/g, '').length;`
これにより、改行文字を取り除いた状態での文字数を計算します。

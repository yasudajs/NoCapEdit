## Verification Plan

### Test Checklist
1. **初回起動フロー**:
   - config.json がない状態で起動し、初期設定ダイアログでホームフォルダが設定できること。
2. **自動ファイル作成**:
   - 起動時およびプラスボタン操作時に `yyyymmdd_hhmmss.txt` 形式で衝突を回避しながらファイルが自動生成されること。
3. **自動保存の動作**:
   - 文字入力停止から3秒後に自動保存されること（右下に邪魔なトースト通知が出ないこと）。
4. **タブ切替・クローズ時の保存/削除**:
   - タブ切替時に未保存内容が保存されること。
   - 空（空白・改行のみ）の自動生成ファイルが、タブクローズ時やアプリ終了時にディスクから自動削除されること。
   - 何かしら入力済みのファイルは削除されず保存されること。
5. **アプリ終了フロー**:
   - 未保存状態でアプリを閉じた際、自動保存が走り終了すること。また必要に応じてリカバリーダイアログが表示されること。

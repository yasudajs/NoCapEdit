# 手動テスト手順 (フェーズ1-1: タイムスタンプ生成ロジック共通化)

本テストは、タイムスタンプの生成ロジックを共通化した影響で、ファイルの新規保存・手動保存時に正しい名前のファイルが生成されることを確認するためのものです。

## テスト項目
- `[ ]` アプリケーション（NoCapEdit）を起動できること
- `[ ]` 初回保存時の確認
  - アプリ起動後、新しいタブを開き、適当なテキストを入力する。
  - ショートカットキー (`Ctrl+S`) またはメニューから保存を実行する。
  - 保存されたファイル名が `yyyyMMdd_HHmmss.nctx` の形式になっていることを確認する。
- `[ ]` 手動保存（別名で保存）時の確認
  - 既存のタブまたは新規タブで、エディタの設定等から手動保存（保存モードの変更や別名保存機能があればそれを使用）を実行する。
  - 手動保存されたファイルが同様に `yyyyMMdd_HHmmss.nctx` 形式になっていることを確認する。

# 選択範囲の文字数カウント機能実装の変更内容（Walkthrough）

選択範囲の文字数をステータスバーの右側へ表示する機能の実装が完了しました。

## 変更内容

### フロントエンド

#### 1. [main.js](file:///c:/work/NoCapEdit/src/dist/main.js)
- **[updateEditorMetrics](file:///c:/work/NoCapEdit/src/dist/main.js#L88) 関数の修正**:
  - `editor.selectionStart` と `editor.selectionEnd` から選択範囲の有無を判定するようにしました。
  - 選択範囲が存在する場合、該当部分のテキストから改行コード（`\r\n` / `\n`）を除外した文字数をカウントし、`選択文字数 / 全体文字数 chars`（例: `5 / 15 chars`）と表示するロジックを実装しました。
  - 選択範囲がない場合は、従来どおり `全体文字数 chars`（例: `15 chars`）と表示します。
- **[setupUIEventListeners](file:///c:/work/NoCapEdit/src/dist/main.js#L745) 関数の修正**:
  - マウスでドラッグしてテキストを選択した際にもステータスバーの表示がリアルタイムで更新されるよう、`editor` に `mouseup` イベントリスナーを追加し、`updateEditorMetrics` が呼び出されるようにしました。

### 仕様書とバージョン管理

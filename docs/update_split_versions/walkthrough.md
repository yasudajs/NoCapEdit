## 検証結果 (手動検証)

GitHub上のリリース一覧をシミュレートするテスト用モックデータ（`v0.2.16`, `v0.1.38`, `v0.1.34`）を用いて、以下の動作確認を行いました。

1. **検証1: v0.1系古いバージョンでの動作確認 (アプリバージョン: `0.1.33` をシミュレート)**
   - 期待値: `v0.2.16` は無視され、`v0.1.38` のアップデート通知（`Ver 0.1.33 → Ver 0.1.38`）が表示されること。
   - 結果: **[合格]** 期待通り、v0.2系のダミーは無視され、v0.1系の最新が検知・表示されました。
2. **検証2: v0.2系古いバージョンでの動作確認 (アプリバージョン: `0.2.15` をシミュレート)**
   - 期待値: `v0.1.38` は無視され、`v0.2.16` のアップデート通知（`Ver 0.2.15 → Ver 0.2.16`）が表示されること。
   - 結果: **[合格]** 期待通り、他系統のリリースは無視され、自身の系統の最新が検知・表示されました。
3. **検証3: 最新バージョンでの非検知確認 (アプリバージョン: `0.2.16`)**
   - 期待値: 起動バージョンが最新タグと一致するため、アップデート通知が表示されないこと。
   - 結果: **[合格]** 期待通り、アップデート通知は表示されませんでした。

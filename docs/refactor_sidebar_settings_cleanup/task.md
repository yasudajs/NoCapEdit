# 手動テストのチェックリスト

以下の操作を行い、機能が正常に動作するか（設定が保存されるか）を確認してください。

- [ ] アプリを起動する
- [ ] サイドバーのトグルボタン（表示/非表示）をクリックし、サイドバーを非表示にする
- [ ] アプリを一度終了し、再度起動する
- [ ] サイドバーが非表示の状態で復元されることを確認する
- [ ] サイドバーのトグルボタンをクリックし、サイドバーを表示させる
- [ ] サイドバーの境界線（リサイズハンドル）をドラッグして、幅を広げる
- [ ] アプリを一度終了し、再度起動する
- [ ] サイドバーが表示状態であり、かつ広げた幅が復元されることを確認する

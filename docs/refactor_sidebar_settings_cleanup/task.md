# 手動テストのチェックリスト

以下の操作を行い、機能が正常に動作するか（設定が保存されるか）を確認してください。

- [x] アプリを起動する
- [x] サイドバーのトグルボタン（表示/非表示）をクリックし、サイドバーを非表示にする
- [x] アプリを一度終了し、再度起動する
- [x] サイドバーが非表示の状態で復元されることを確認する
- [x] サイドバーのトグルボタンをクリックし、サイドバーを表示させる
- [x] サイドバーの境界線（リサイズハンドル）をドラッグして、幅を広げる
- [x] アプリを一度終了し、再度起動する
- [NG] サイドバーが表示状態であり、かつ広げた幅が復元されることを確認する
    → 復元されないが、今回はこの仕様でOKとする。

## User Review Required

> [!IMPORTANT]
> **四隅からの Flood Fill 透過処理**:
> 単純な白色透過を行うと中央の `nce` という白文字まで透明になってしまいます。
> そのため、画像の四隅（左上、右上、左下、右下）から白色を検出して隣接ピクセルを探索・透明化する「Flood Fill（ペイント缶塗りつぶしと同等）」のアルリズムを実行し、ロゴ文字を保護しながら角の白色だけを正確に透明にします。

- `[ ]` ユーザーによる手動テストの実施
  - **1. アプリ起動時の挙動確認**
    - `[ ]` 保存モード（自動保存）で起動した場合、従来通りファイルが作成され、タブ名が表示されること。
    - `[ ]` 無保存モードで起動した場合、ファイルは作成されず、タブ名が `[yyyy/mm/dd hh:mm:ss]` になり、ステータスバー左下に `[無保存モード]` と表示されること。
  - **2. モード切り替えの挙動確認**
    - `[ ]` 設定画面でモードを切り替えた際、即座にステータスバーの表示が切り替わること。
    - `[ ]` 「自動保存」から「無保存」に切り替えた場合、以降の文字入力で自動保存が走らないこと。
  - **3. 無保存モードでの終了・タブ閉じ時の挙動確認**
    - `[ ]` 無保存モードで文字を入力して未保存状態にする。タブを閉じた際に、確認ダイアログが出ず、内容が保存されずに消えること。
    - `[ ]` アプリを終了した際に、同様に内容が破棄されて正常終了すること。
  - **4. 無保存モードでの手動保存（Ctrl+S）の挙動確認**
    - `[ ]` 無保存モードで Ctrl+S を押した際、ホームフォルダに `yyyymmdd_hhmmss.nctx` というファイルがダイアログなしで直接作成され、入力内容が保存されていること。
    - `[ ]` 手動保存した後もタブの表示は `[yyyy/mm/dd hh:mm:ss]` のままであり、ステータスバーも `[無保存モード]` のままであること。
    - `[ ]` 手動保存後にさらに追記してタブを閉じた場合、追記分はファイルに保存されず破棄されること。

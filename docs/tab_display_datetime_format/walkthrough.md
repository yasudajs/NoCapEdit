# タブ表示名のフォーマット対応（日時表示・拡張子非表示化） 修正内容の確認

タブに表示されるファイル名を、自動生成されたファイルについては日時形式 `yyyy/mm/dd hh:mm:ss` に変換し、それ以外のファイルについては拡張子を非表示にする対応を行いました。

## 変更内容

### フロントエンド (JavaScript)

#### [main.js](file:///c:/work/NoCapEdit/src/dist/main.js)
- 自動生成されたファイル名（例: `20260709_222843.nctx`）をパースし、`2026/07/09 22:28:43` のように表示するためのヘルパー関数 `formatTabDisplayName(fileName)` を実装しました。
- 同名ファイルが回避されて末尾にインデックスがある場合（例: `20260709_222843_01.nctx`）は、インデックスを数値に変換して `2026/07/09 22:28:43-1` のようにハイフンで繋ぎます。
- 自動生成以外のファイル（例: `memo.txt`）は、最後の拡張子を取り除いて表示するようにしました。ただし隠しファイルなど（ドットが先頭文字の場合）は元の名前を維持します。
- `renderTabs` 関数のタブ描画処理において、タブ名テキストにこのフォーマット関数を適用するように変更しました。

## 検証結果

Node.js 環境にて、多様なファイル名パターンを用いた単体テストを実施し、すべての条件が正しくフォーマットされることを確認しました。
